Creates a new ServiceMessage packet.

Note that this does not send the message. ServiceMessage is only intended to represent a BSI packet.

Destination is multi-cast by default. However for uni-cast specify the target destination as a single element in the destinations list.

Example:

var src = this.reference;
var dest = getDestination();
var msg = getMessage();

ServiceMessage(source: src, destinations: [dest], message: msg);

For broadcast messages, use ServiceMessage.broadcast.

Example:

var src = this.reference;
var msg = getMessage();

ServiceMessage.asBroadcast(source: this, message: msg);
